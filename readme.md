# Esuts Flask

:)

## Install

```shell
npm i esuts-flask
```

## Usage

```coffeescript
$ = require 'esuts-flask
$.info $.now()
```

## Method

- $.delay_([time])
- $.each()
- $.extend()
- $.get_(url, [data])
- $.i(msg)
- $.info([method], [type], msg)
- $.log(msg)
- $.noop()
- $.now()
- $.param()
- $.parseJSON()
- $.parseJson(input)
- $.parseString(input)
- $.post_(url, [data])
- $.serialize(string)
- $.timestamp([input])
- $.trim()
- $.type(input)
